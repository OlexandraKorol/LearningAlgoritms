## Завдання 1:

Було створено масив чисел від `0` до `100000` з кроком `10`. Кожен алгоритм виконувався `100000` разів для точнішого вимірювання часу.

Результат виконання у консолі:

```text
Interpolation: 5140208 ns
Binary: 15619625 ns
```


У цьому тесті `Interpolation Search` спрацював швидше за `Binary Search`, оскільки масив рівномірно відсортований. Інтерполяційний пошук “вгадує” приблизну позицію елемента і тому виконує менше кроків.
